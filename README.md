# Human Verification Experiment

Experimental Next.js app for human-vs-automation verification using timing, input behavior, image interpretation, drawing, and CAPTCHA-like interaction signals.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment

Copy `.env.example` to `.env` and fill in local values. Do not commit `.env` or `.env.local`.

## Build

```bash
npm run build
```
